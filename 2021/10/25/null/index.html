Go语言快速入门

需求

开发hello.go程序，输出 “hello,world”

开发步骤

1、安装软件

2、创建目录

3、新建文件

4、开发代码

  • go程序的后缀是.go
  • package main 文件所在的包为main。在go中，每个文件都归属一个包。
  • import “fmt” 表示引入包，可以使用fmt函数,例如 fmt.Println(“这可以打印”)
  • func main，func表示关键词，函数。main是函数名，是一个主函数，是我们函数的入口

编译和执行

  • 编译命令 go build 文件名 —->> ./文件名
  • 直接运行 go run 文件名

两者的区别

  • 可执行文件可以拷贝到没有开发环境的机器上执行
  • go run则需要开发环境
  • 可执行文件变大，因为包含所依赖的库文件

注： go build -o hello_rename.exe hello.go 可以自己定义名字

ctrl + d : 复制代码 ctrl + y : 删除代码

GO开发注意事项

  1. go必须是.go后缀名才能编译
  2. 执行入口是main()函数
  3. Go是区分大小写的
  4. 每个语句不需要加分号，程序自己加分号
  5. 编译是一行一行编译的，一行就写一行语句
  6. Go语言自定的包或者import的包没有用到，会报错
  7. 大括号成对出现，缺一不可

Go语言转义字符

\t 制表符，实现对齐功能 \n 换行符 \r 回车

注释方式

注释可以提高代码的阅读性

行注释： // 注释内容

块注释： /**/

注释快捷键：ctrl + /

说明：被注释的代码不会被执行，且块注释里不能嵌套块注释

规范和代码风格

  • 官方建议使用行注释
  • 要有正确的缩进和空白 tab键可缩进（shift + tab 整体向左移动）（tab 整体向右移动）
  • 运算符的两边建议各加空格（a = b + c）
  • 代码风格一致
  • 每行不要超过80个字，可以逗号换行，加号换行

官方网站：www.golang.org

字符类型

Golang没有专门的字符类型，如果要存储单个字符（字母），一般使用byte来保存。

Go的字符串不同，它是有字节组成的

  • 如果我们保存的字符在ASCII表中，比如[0-1,a-z,A-Z]直接可以保存到byte
  • 如果我们保存的字符对应码值大于255，这时我们可以考虑使用uint类型保存
  • 如果我们需要按字符的方式输出，这时我们需要使用格式化输出，fmt.Printf(“%c”,c1)。。。

字符类型使用的细节

  • 字符常量是用单引号括起来的单个字符
  • Go中允许使用转义字符’\‘来将其后的字符转变为特殊字符型常量
  • Go语言的字符使用UTF-8编码
  • Go中，字符本质是一个整数，直接输出时，时该字符对应的UTF-8编码的码值
  • 可以直接给某一变量赋一个数字，然后按格式化输出时%c会输出该数字对应的unicode字符
  • 字符类型可以进行运算，可以当成数字，因为它有对应有Unicode码

布尔类型

  • 只允许取true或者false,不能用0或者非0来表示 bool类型
  • 只占1个字节
  • 适用于逻辑运算，一般用于程序流程控制

字符串类型

基本使用：

  • Go语言的字符串采用UTF-8编码，这样Golang统一了编码，中文乱码问题不会困扰程序员
  • Golang的字符串是不可变的
  • 字符串的表示形式有两种：双引号（识别转义字符） 反引号（字符串的原生形式输出,包括换行和特殊字符，可实现防止攻击，输出源代码的效果）
  • 字符串拼接用加号

标识符

概念

  • Golang对各种变量、方法、函数等命名时使用的字符序列称为标识符
  • 凡是自己可以起名字的地方都叫标识符

命名规则

  • 由26个英文字母大小写、0-9、_组成
  • 数字不能开头
  • 严格区分大小写
  • 标识符不能包含空格
  • 下划线“_”本身就是一个特殊的标识符，称为空标识符。可以代表任意其他的标识符，但它对应的值会被忽略。所以仅能作为占位符使用，不能作为标识符使用
  • 不能以系统保留关键字作为标识符，例如：break ,if 等

注意事项

  • 包名：尽量和路径目录保持一致，尽量采取有意义的包名，简短，有意义，不要和系统库的包名冲突
  • 变量名、函数名、常量名：采用驼峰法
  • 如果变量名、函数名、常量名首字母大写，则可以被其他包访问，如果首字母小写，则只能在本包使用

switch结构

  • case后边是一个表达式 常量、变量、有返回值的函数也可以
  • case后的各个表达式的值的数据类型，必须和switch的数据类型一致
  • case后面可以带多个表达式，使用逗号分割，例如：case 表达式1，表达式2
  • case后面的表达式如果是常量，则要求不能重复（变量检测不出来）
  • case后面不需要带break,程序匹配到一个case后会执行对应的代码块。然后退出switch
  • default语句不是必须的
  • switch后可以不带表达式，类似多个if-else分支来使用
  • switch后也可以直接声明/定义一个变量，分号结束，但是不推荐
  • switch穿透fallthrough 如果在case语句块后增加fallthrough，则会继续执行下一个case，也叫switch穿透
  • Type Switch : switch语句还可以被用于type-switch来判断某个interface变量中实际指向的变量类型

函数

函数注意事项和细节讨论：

  • 函数的形参列表可以时多个，返回值列表也可以时多个
  • 形参列表和返回值列表的数据类型可以是值类型和引用类型
  • 函数的命名遵循标识符命名规则，首字母不能是数字，首字母大写该函数可以被本包文件和其他包文件使用，类似public,首字母小写的话只能被本包文件使用，其他包文件不能使用，类似private
  • 函数中的变量是局部的，函数外不生效
  • 基本数字类型和数组默认的是值传递，即进行值拷贝，在函数内修改是不会影响原来的值
  • 如果希望函数内的变量能修改函数外的变量，可以传入变量的地址&，函数内以指针的方式操作变量，从效果上看是引用
  • Go函数不支持重载
  • Go中，函数也是一种数据类型，可以赋值给一个变量，则该偏凉就是一个函数类型的变量。通过该变量可以对函数调用
  • 函数既是一种数据类型，因此在Go中，函数可以作为形参，并且可调用
  • 为了简化函数类型定义，Go支持自定义数据类型

基本语法： type 自定义数据类型 数据类型 //相当于别名

  • 支持对函数返回值命名
  • 使用_标识符，可忽略返回值
  • Go支持可变参数

init函数

每一个源文件都可以包含一个init函数，该函数会在main函数执行前，被Go运行框架调用，也就是说init会在main函数前被调用

如果一个文件同时包含全局变量定义，init函数和main函数，则执行的流程是变量定义->init函数->main函数

init函数的最主要的作用就是完成一些初始化工作。

字符串函数

常用的字符串函数

  • 统计字符串的长度，按字节len(str),一个汉字是3个字节的,字母和数字是一个字节的
  • 字符串遍历，同时处理有中文的问题 r :=[]rune(str)
  • 字符串转整数：n,err := strconv.Atoi(“12”)
  • 整数转字符串：str = strconv.Itoa(12345)
  • 字符串转[]byte:var bytes = []byte(“hello go”)
  • []byte 转字符串：str = string([]byte{97,98,99})
  • 10进制转2，8，16进制：str=strconv.FormatInt(123,2) // 2->8,16
  • 查找字串是否在指定的字符串中：strings.Contains(“seafood”,”foo”) //true
  • 统计一个字符串有几个指定的字串：strings.Count(“ceheese”,”e”) //4
  • 不区分大小写的字符串比较（==是区分字母大小写的）：fmt.Println(strings.EqualFold(“abc”,”Abc”)) //true
  • 返回子串在字符串第一次出现的index值，如果没有返回-1：strings.Index(“NLT_abc”,”abc”) //4
  • 返回子串在字符串最后一次出现的index,如果没有返回-1 ：strings.LastIndex(“go golang”,”go”)
  • 将指定的子串替换成另外一个子串：strings.Replace(“go go hello”,”go”,”go语言”,n) n可以指定你希望替换几个，如果n=-1表示全部替换
  • 按照指定的某个字符，为分割标识，将一个字符串拆分字符串数组：strings.Split(“hello,world,ok”,”,”)
  • 将字符串的字母进行大小写的转换，strings.ToLower(“GO”) // strings.ToUpper(“go”)
  • 将字符串左右的空格去掉：strings.TrimSpace(“ tn a lone gopher ntrn “)
  • 将字符串左右两边指定的字符去掉，strings.Trim(“!hello!”,”!”)
  • 将字符串左边指定的字符去掉：strings.TrimLeft(“!hello!”,”!”)
  • 将字符串右边指定的字符去掉：strings.TrimRight(“!hello!”,”!”)

使用细节和注意事项

  • 数组是多个$\textcolor{red}{相同类型} $的数据的组合，一个数组一旦声明/定义了，其长度是固定的，不能动态变化
  • var arr []int 这时arr就是一个slice切片
  • 数组中的元素可以是任意数据类型，包括值类型和引用类型，但是不能混用
  • 数组创建后，如果没有赋值，有默认值（数值类型数组：0 字符串数组：“” bool类型数组：false）
  • 使用数组的步骤：1、声明数组并开辟空间 2、给数组各个元素进行赋值 3、使用数组
  • 数组的下标是从0开始的
  • 数组下标必须在指定范围内使用，否则报panic；
  • Go的数组是值类型，在默认情况下是值传递，因此只能进行值拷贝。数组间不会相互影响
  • 如想在其他函数中，去修改原来的数组，可以使用引用传递（指针方式）
  • 长度是数组类型的一部分，在传递函数参数时，需要考虑数组的长度

切片

基本介绍

  • 切片的英文时slice
  • 切片是数组的引用，因此切片是引用类型，在进行传递的时候，遵守引用传递的机制
  • 切片的使用和数组类似，遍历切片、访问切片的元素和求切片长度都一样
  • 切片的长度是可以变化的，因此切片是一个可以动态变化的数组
  • 切片定义的基本语法：

var 变量名 []类型 例如： var a []int

切片的使用

  • 定义一个切片，然后让切片引用一个已经创建好的数组，
  • 通过make来创建切片

基本语法：var 切片名 []type = make([]tpye,len,[cap])

  • 定义一个切片，直接就指定具体数组，使用原理类似make的方式

方式一和方式二的区别：

方式一：直接引用数组，这个数组是事先存在的，程序员是可见的

方式二：通过make来创建切片，make也会创建一个数组，是由切片在底层进行维护，程序员是看不见的

切片注意事项和细节说明

  • 切片初始化时，var slice = arr[startIndex:endIndex]

说明，不含endIndex

  • 切片初始化时，不能越界，范围在[0-len(arr)]之间，但是可以动态增长
  • var slice = arr [0:end] 可以简写为 var slice = arr[:end]
  • var slice = arr[start:len(arr)] 可以简写为 var slice = arr[start:]
  • var slice = arr[0:len(arr)] 可以简写为 var slice = arr[:]
  • cap是一个内置函数，用于统计切片的容量
  • 切片定义完成后，还不能使用，本身是空的，需要让其引用到一个数组
  • 切片可以继续切片
