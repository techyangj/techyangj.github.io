vi与vim

在很多的Linux发行版中，直接把vi做成vim的软连接

vi是Linux中最经典的文本编辑器
vi的设计核心思维：让程序员的手指始终保持在键盘的核心区域，就能完成所有的编辑操作
vi的特点：
没有图形界面的功能强大的编辑器
只能编辑文本内容，不能对字体、段落进行排版
不支持鼠标操作，没有菜单，只有命令
vi编辑器在系统管理、服务器管理编辑文件时，其功能永远不是图形界面的编辑器所比拟的

vim

vim是vi发展出来的一个文本编辑器，支持代码补全、编译及错误跳转等方便编程的功能特别丰富，在程序员中被广泛使用，被称为编辑器之神

打开和新建文件

如果文件已经存在，会直接打开该文件
如果文件不存在，会新建一个文件

打开文件且定位行

如果只带上+而不指定行号，会直接定位到文件末尾

异常处理

如果vi异常退出，在磁盘上可能会保存有交换文件
下次再使用vi编辑该文件时，会出现提示信息，按下字母D可以删除交换文件

vim工作模式

vim有三种工作模式：命令模式、末行模式、编辑模式

1、命令模式
打开文件即可进入命令模式
通过命令可对文件进行常规的编辑操作，例如：定位、翻页、复制、粘贴、删除…
2、末行模式
末行模式时文件的出口，要返回控制台必须在末行模式下输入命令
3、编辑模式（正常编辑文字）

末行模式命令

命令 英文 功能
w write 保存
q quit 退出，如果没有保存，不允许退出
q! quit 强行退出，不保存退出
wq write&quit 保存并退出
x 保存并退出

常用命令

移动

移动在命令模式下

命令 功能 手指
h 向左 食指
j 向下 食指
k 向上 中指
l 向右 无名指
命令 功能
Home键、^、数字0 跳转到行首
End键、$键 跳转到行尾
PgUp键、PgDn 向上翻页、向下翻页
1G或gg 跳转到文件的首行
G 跳转到文件的末尾行
操作类型 指令 用途
复制 yy、#yy 复制光标处的一行、#行
粘贴 p、P 粘贴到光标处之后、之前
删除 x或Delete 删除光标处的单个字符
删除 dd、#dd 删除光标处的一行、#行
删除 d^ 从光标处之前删除至行首
删除 d$ 从光标处删除至行尾
删除 C 从光标处删除至行尾并且进入插入模式
操作类型 指令 用途
文本查找 /word 向后查找字符串”word”
文本查找 n、N 跳转后/前一个结果
撤销编辑 u 撤销最近的一次操作
撤销编辑 U 撤销对当前行的所有修改
撤销编辑 Ctrl + r 取消前一次撤销操作
保存退出 ZZ 保存修改并退出
字符串替换
操作类型 指令 用途
行间替换 : s /old/new 替换当前行第一个”old”
行间替换 ：s /old/new 替换当前行所有”old”
区域内替换 :n,m s/old/new/g 替换第n-m行所有的”old”
区域内替换 :% s/old/new/g 替换文件内所有的”old”

开关参数设置
操作类型 | 指令 | 用途
———|———-|———
编辑器设置 | :set nu|nonu | 显示/不显示行号
编辑器设置 | :set ai|noai | 启动、关闭自动缩进
