首先win+R打开终端黑窗口，之后在黑窗口中输入：
node -v 和 npm -v 查看版本信息

安装hexo

cnpm install -g hexo-cli

查看是否安装成功：
hexo -v

打开博客

打开博客的命令为:hexo s

通过打开本地网址，就可访问自己的博客了。

断开连接：ctrl+c

hexo基础命令

1、hexo init 初始化博客
2、hexo g 生成博客
3、hexo clean 清除缓存文件
4、hexo s 启动博客
5、hexo d 将博客推入远端
6、hexo new “文章名” 创建一个博客
这几个命令是hexo中常见的命令
