Poetry
======
## Poetry 安装
```
  pip install poetry  # 安装poetry
```
或者脚本安装： https://python-poetry.org/docs/
## 相关指令
```
  poetry init         # 初始化 
  poetry shell        # 进入环境
  poetry add XXX      # 添加依赖
```
