变量和数字

变量：matlab的变量区分大小写，可以为字母，自动创建变量，并且分配适当大小的存储，变量名称可以为任意长度，但是建议取短一点。
数字： 浮点数的有限精度为16位有效小数位数。

矩阵运算符
表达式 作用
+ 加法
- 减法
* 乘法
/ 除法
\ 左除
^ 幂
‘ 复共轭转置
() 指定计算顺序
数组运算符
表达式 作用
+ 加法
- 减法
.* 逐元素乘法
./ 逐元素除法
.\ 逐元素左除
.^ 逐元素幂
.’ 非共轭数组转置
函数

matlab内置大量的数学函数，例如abs、sqrt、exp、sin等
内置常量值：pi、i、j、eps、realmin、realmax、Inf(无穷大)、NaN(非数字)

    format short
    format long
    format bank
    format rat

    等函数
  • 隐藏输出

  • 输入长语句
    一行无法容纳，可使用输入长语句，用三个…后跟enter以指示该语句在下一行继续。
